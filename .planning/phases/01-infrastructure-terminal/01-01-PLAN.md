         - No pagination needed for candles endpoint (per research: returns all rows)

         - Redis cache key: `moex:dividends:${ticker}`, TTL 604800 (7 days)
         - URL: `${baseUrl}/securities/${ticker}/dividends.json?iss.meta=off`
         - Parse: `json.dividends.columns` + `json.dividends.data` -> DividendData[]
       - `getHistoryWithPagination(url)`:
         - For INFR-03: history endpoints paginate at 100 rows
         - Loop: fetch URL with `start` param, check `history.cursor` data block for `[INDEX, TOTAL, PAGESIZE]`
         - Accumulate `history.data` rows until INDEX + PAGESIZE >= TOTAL
         - Return all accumulated rows
       - Import `redis` from `@/lib/redis` for caching
