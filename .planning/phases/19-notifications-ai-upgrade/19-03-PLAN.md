    Add a static method `formatForChat` to `GridAiService` that converts a `GridSuggestion` into a human-readable text block that can be injected into AI chat context (AI-06).

    ```typescript
    static formatForChat(suggestion: GridSuggestion, instrument: string): string {
      const step = ((suggestion.upperPrice - suggestion.lowerPrice) / suggestion.gridLevels)
      return [
        `Предложение Grid Trading для ${instrument}:`,
        `Диапазон: ${suggestion.lowerPrice} — ${suggestion.upperPrice}`,
        `Уровней сетки: ${suggestion.gridLevels}`,
        `Шаг сетки: ${step.toFixed(4)}`,
        `Распределение: ${suggestion.gridDistribution}`,
        `Сумма на ордер: ${suggestion.amountPerOrder}`,
        `Комиссия: ${(suggestion.feeRate * 100).toFixed(2)}%`,
        `Ожидаемая прибыль на сетку: ${suggestion.expectedProfitPerGrid.toFixed(2)}%`,
        `Примерное кол-во сделок/месяц: ${suggestion.estimatedMonthlyTrades}`,
        ``,
        `Обоснование: ${suggestion.reasoning}`,
      ].join("\n")
    }
    ```

    This method is designed to be called from the chat route when user requests grid analysis for a specific instrument. The formatted text gets prepended to messages as system context, similar to how AiContextService.assembleContext works.

    In the chat route (`src/app/api/ai/chat/route.ts` — already modified in Plan 02), the integration point is:
    - When `context?.ticker` is provided AND user message mentions "grid"/"сетка"/"грид", call `GridAiService.suggestParams` and prepend `formatForChat` result to enriched messages.
    - This wiring will be done in the chat route as a final step after Plan 02 changes are in place.

    However, since Plan 02 already modifies `route.ts`, add this integration there too. In `route.ts` after the context enrichment block:
    ```typescript
    // Grid AI suggestion integration
    const lastUserMsg = safeMessages.filter(m => m.role === "user").pop()?.content?.toLowerCase() ?? ""
    const gridKeywords = ["grid", "грид", "сетка", "сеточн"]
    const wantsGrid = gridKeywords.some(kw => lastUserMsg.includes(kw))

    if (wantsGrid && context?.ticker && context?.figi) {
      try {
        const { GridAiService } = await import("@/server/services/grid-ai-service")
        const suggestion = await GridAiService.suggestParams({
          instrumentId: context.figi,
          instrument: context.ticker,
          userId,
        })
        const gridContext = GridAiService.formatForChat(suggestion, context.ticker)
        enrichedMessages = [
          { role: "user", content: `[Grid Trading анализ]\n${gridContext}` },
          ...enrichedMessages,
        ]
      } catch (e) {
        console.error("[AI Chat] Grid suggestion failed:", e)
      }
    }
    ```
