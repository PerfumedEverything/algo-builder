**Create `src/server/providers/broker/bybit-provider.ts`:**

Class `BybitProvider` implements `BrokerProvider`:

**connect(token):**
- Parse token as `"apiKey:apiSecret"` format — split on first `:` to handle secrets with colons
- `const [key, ...rest] = token.split(':'); const secret = rest.join(':')`
- Create `this.client = new RestClientV5({ key, secret, testnet: true })` (per D-03, D-14)

**disconnect():** Set `this.client = null`

**getAccounts():**
- Call `getWalletBalance({ accountType: 'UNIFIED' })`
- Return `[{ id: 'bybit-unified', name: 'Bybit Unified', type: 'BYBIT', openedDate: '' }]`

**getPortfolio(accountId):**
- Parallel: `getWalletBalance({ accountType: 'UNIFIED' })` + `getPositionInfo({ category: 'linear', settleCoin: 'USDT' })`
- Map wallet: `totalEquity`, `totalPerpUPL`, `totalAvailableBalance` (all parseFloat from string)
- Map positions: for each position, use `unrealisedPnl`, `avgPrice`, `markPrice`, `size` directly from API (per D-02, no custom P&L math)
- `instrumentType: 'CRYPTO'` for all positions
- Return `Portfolio` object following pattern from RESEARCH.md Pattern 2

**getCandles(params):**
- Determine category: `'linear'` for USDT perps (per D-02)
- Call `getKline({ category, symbol, interval: BYBIT_INTERVAL_MAP[params.interval], start: params.from.getTime(), end: params.to.getTime(), limit: 1000 })`
- **CRITICAL:** `.reverse()` the result array (Bybit returns newest-first, per Pitfall 3)
- Map each `[startTime, open, high, low, close, volume, turnover]` to `Candle` type
- All numeric fields are strings — `parseFloat()` each (per anti-pattern)

**getCurrentPrice(instrumentId):**
- Call `getTickers({ category: 'linear', symbol: instrumentId })`
- Return `parseFloat(result.list[0].lastPrice)`

**getAvailableCash(accountId):**
- Call `getWalletBalance({ accountType: 'UNIFIED' })`
- Return `parseFloat(account.totalAvailableBalance)`

**placeOrder(params):**
- Call `submitOrder({ category: 'linear', symbol: params.symbol, side: params.side === 'BUY' ? 'Buy' : 'Sell', orderType: params.orderType === 'LIMIT' ? 'Limit' : 'Market', qty: String(params.quantity), price: params.price ? String(params.price) : undefined, timeInForce: 'GTC' })`
- Return `res.result.orderId`

**cancelOrder(orderId, symbol):**
- Call `cancelOrder({ category: 'linear', symbol, orderId })`

**getInstruments(type):**
- If type is 'CRYPTO': return `BYBIT_CRYPTO_PAIRS.map(p => ({ figi: p.symbol, ticker: p.symbol, name: p.name, type: 'CRYPTO', currency: 'USDT', lot: 1 }))`
- Otherwise return `[]`

**Private helper:** `ensureClient(): RestClientV5` throws if not connected.

Keep file under 150 lines — extract helpers to bybit-constants.ts.
