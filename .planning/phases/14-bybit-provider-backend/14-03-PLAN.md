**Replace `evaluateCrossing` function:**

Old approach (stateful):
```typescript
evaluateCrossing(type, current, target, indicatorKey, lastValues)
// Needs lastValues map persisted across check cycles
```

New approach (stateless, batch):
```typescript
import { Indicators } from '@ixjb94/indicators'

const ta = new Indicators()

export const evaluateCrossingBatch = async (
  type: "CROSSES_ABOVE" | "CROSSES_BELOW",
  series: number[],
  target: number,
): Promise<boolean> => {
  if (series.length < 2) return false
  const result = type === "CROSSES_ABOVE"
    ? await ta.crossOverNumber(series, target)
    : await ta.crossUnderNumber(series, target)
  return result[result.length - 1] ?? false
}
```

**Update `evaluateCondition`:**
- Change signature to accept `candles` directly (not just `ctx` with price)

  2. Call `evaluateCrossingBatch(condition.condition, series, target)`
  3. Return result
