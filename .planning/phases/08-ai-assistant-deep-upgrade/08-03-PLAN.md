  <action>
    1. Create `src/hooks/use-ai-stream.ts` — a custom hook that manages streaming AI chat:

    2. Hook signature: `useAiStream(params: { onStrategyExtracted?: (s: AiGeneratedStrategy) => void })`

    3. State managed by hook:
       - `messages: ChatMessage[]` — full conversation history
       - `streamingContent: string` — current streaming text being built
       - `thinkingContent: string` — current thinking text (shown as indicator)
       - `isStreaming: boolean` — whether currently streaming
       - `isThinking: boolean` — whether in thinking phase

    4. ChatMessage type (extended from AiChatMessage):
       ```
       type ChatMessage = AiChatMessage & {
         strategy?: AiGeneratedStrategy
         actions?: QuickAction[]
         hidden?: boolean
         isStreaming?: boolean
         thinkingContent?: string
       }
       ```

    5. Core method `sendMessage(text: string, options?: { hidden?: boolean; forceCreate?: boolean; context?: { ticker?: string; timeframe?: string; figi?: string } })`:
       - Add user message to messages array
       - Set isStreaming=true
       - Call `fetch("/api/ai/chat", { method: "POST", body: JSON.stringify({ messages: chatHistory, context: options?.context, forceCreate: options?.forceCreate }) })`
       - Read response.body as ReadableStream
       - Parse SSE chunks:
         - `type: "thinking"` -> append to thinkingContent, set isThinking=true
         - `type: "content"` -> append to streamingContent, set isThinking=false
         - `type: "strategy"` -> parse JSON, store as pendingStrategy
         - `type: "done"` or `[DONE]` -> finalize: create assistant ChatMessage from accumulated content, attach strategy if present, attach QuickAction[] if strategy was proposed, add to messages[], reset streaming state
       - On error: add error assistant message

    6. When strategy is proposed (type: "strategy" chunk received), inject standard QuickAction buttons:
       ```
       actions: [
         { label: "Создать эту стратегию", action: "CREATE" },
         { label: "Покажи другие варианты", action: "MORE" },
         { label: "Изменить риски", action: "ADJUST_RISKS" },
       ]
       ```

    7. Method `resetChat()` — resets messages to initial greeting, clears all streaming state.

    8. Method `addSystemMessage(text: string)` — appends an assistant message (used for "strategy created" markers per AIUP-09).

    9. Return: `{ messages, streamingContent, thinkingContent, isStreaming, isThinking, sendMessage, resetChat, addSystemMessage }`
  </action>
