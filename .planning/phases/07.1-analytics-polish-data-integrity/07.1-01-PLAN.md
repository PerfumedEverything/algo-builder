    **Step 3: Create portfolio-health-service.ts** as a class with static pure methods (per project OOP convention for services, but static since no state/deps):
    - `static computeDiversificationScore(concentration: ConcentrationIndex, sectorAllocation: SectorAllocation[]): HealthSubScore`
      HHI scoring: hhi < 0.10 = 100, < 0.15 = 80, < 0.25 = 50, >= 0.25 = 20
      Sector count bonus: >= 4 sectors = +10, >= 6 = +20
      Dominant position penalty: any in concentration.dominantPositions = -30
      Clamp 0-100. Details array: human-readable strings explaining each factor.

    - `static computeRiskScore(highCorrelationCount: number): HealthSubScore`
      Base 80 (no portfolio volatility metric available without extra API calls).
      Each high-correlation pair: -5 (max -25).
      Clamp 0-100. Details: count of high-corr pairs, risk level text.

    - `static computePerformanceScore(benchmark: BenchmarkComparison | null, depositRate: number): HealthSubScore`
      If no benchmark: return { score: 50, details: ["Нет данных для сравнения"] }
      Pro-rate deposit: depositRateForPeriod = depositRate * (benchmark.period / 365)
      delta = benchmark.portfolioReturn - benchmark.benchmarkReturn
      delta > 5 = 100, > 0 = 80, > -5 = 50, <= -5 = 20
      If portfolioReturn < depositRateForPeriod * 100 => -20 penalty
      Clamp 0-100.

    - `static computeHealthScore(params: { concentration, sectorAllocation, benchmark, highCorrelationCount, positionCount }): HealthScore`
      If positionCount < 2: return insufficient_data state (total 0).
      Compute 3 sub-scores, weighted sum, round to integer.
      Level: >= 75 excellent, >= 50 good, >= 25 warning, < 25 danger.

    - `static generateDiversificationAdvice(positions: PortfolioPosition[], concentration: ConcentrationIndex, sectorAllocation: SectorAllocation[]): DiversificationAdvice[]`
      Rules per research Pattern 2: position > 30% => danger, sector > 50% => warning, < 3 sectors => warning, missing major sector => warning (check MAJOR_SECTORS), HHI > 0.25 => warning.
      If none triggered => single success "Хорошая диверсификация".

    - `static generateCorrelationWarnings(highPairs: CorrelationMatrix["highPairs"]): CorrelationWarning[]`
      For each pair: lookup FUNDAMENTALS_MAP for sectors, same sector => "(оба SECTOR_LABEL)" in text.
      corr > 0.7: "TICKER_A и TICKER_B движутся вместе REASON — это увеличивает риск"
      corr < -0.7: "TICKER_A и TICKER_B движутся противоположно — это хорошо для диверсификации"
      Return CorrelationWarning[].

    - `static enhanceBenchmark(benchmark: BenchmarkComparison): EnhancedBenchmarkComparison`
      Pro-rate deposit: depositRateForPeriod = DEPOSIT_RATE_ANNUAL * (benchmark.period / 365)
      Compare portfolioReturn (already a percentage like 5.0) vs depositRateForPeriod * 100 (convert to %).
      Verdict logic: if portfolioReturn > benchmarkReturn AND > depositPct => "beats_market" with "Вы обгоняете рынок на {delta}%"
      If portfolioReturn > depositPct but < benchmarkReturn => "beats_deposit" with "Лучше депозита, но рынок обгоняет вас"
      If portfolioReturn < depositPct => "loses_to_deposit" with "Депозит выгоднее на {diff}%"
