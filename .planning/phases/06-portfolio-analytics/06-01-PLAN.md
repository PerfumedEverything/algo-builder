### Task 2: Portfolio Analytics Service
**New file:** `src/server/services/portfolio-analytics-service.ts`

**Class:** `PortfolioAnalyticsService`

**Methods:**
1. `getCorrelationMatrix(userId: string): Promise<CorrelationMatrix>`
   - Get portfolio positions â†’ extract unique tickers
   - Fetch 90 days of daily candles per ticker via BrokerService
   - Calculate daily returns for each ticker
   - Compute pairwise Pearson correlation using `sampleCorrelation` from simple-statistics
   - Return `{ tickers: string[], matrix: number[][], highPairs: { a: string, b: string, corr: number }[] }`
   - highPairs = pairs where |corr| > 0.7

2. `getSectorAllocation(positions: PortfolioPosition[]): SectorAllocation[]`
   - For each position: look up sector from `FUNDAMENTALS_MAP`
   - Calculate market value per position: `quantity * currentPrice`
   - Group by sector, sum values
   - Return `{ sector: string, value: number, percent: number, tickers: string[] }[]`

3. `getAssetTypeBreakdown(positions: PortfolioPosition[]): AssetTypeBreakdown[]`
   - Group positions by `type` (share, etf, bond, currency)
   - Sum market values per type
   - Return `{ type: string, label: string, value: number, percent: number, count: number }[]`

4. `getTradeSuccessBreakdown(userId: string): Promise<TradeSuccessBreakdown>`
   - Get all strategies with operations stats
   - Count profitable vs unprofitable closed strategies
   - Sum profitable P&L vs unprofitable P&L
   - Return `{ profitable: { count, totalPnl }, unprofitable: { count, totalPnl } }`
