# Plan 06-02: Analytics UI Components + Portfolio Integration

## Goal
Correlation heatmap, sector donut, asset type chart, trade success chart — all integrated into portfolio page.

## Tasks

### Task 1: Correlation Heatmap Component
**New file:** `src/components/portfolio/correlation-heatmap.tsx`

- Custom Tailwind grid (N×N for N tickers)
- Color scale: red (-1) → white (0) → green (+1)
- High-correlation pairs (>0.7) get border highlight
- Ticker labels on axes
- Tooltip on hover showing exact correlation value
- Props: `{ matrix: CorrelationMatrix, loading: boolean }`

### Task 2: Sector Allocation Donut
**New file:** `src/components/portfolio/sector-donut.tsx`

- Recharts `PieChart` + `Pie` with donut hole
- Sector colors: predefined map (energy=amber, finance=blue, tech=violet, metals=slate, etc.)
- Legend with sector name + % + value
- Props: `{ data: SectorAllocation[], loading: boolean }`

### Task 3: Asset Type Chart
**New file:** `src/components/portfolio/asset-type-chart.tsx`

- Recharts `BarChart` horizontal — share/etf/bond/currency bars
- Show count and value for each type
- Props: `{ data: AssetTypeBreakdown[], loading: boolean }`

### Task 4: Trade Success Chart
**New file:** `src/components/portfolio/trade-success-chart.tsx`

- Two stats cards: profitable (green) vs unprofitable (red) with count + total P&L
- Simple Recharts `PieChart` showing profitable vs unprofitable by count
- Props: `{ data: TradeSuccessBreakdown, loading: boolean }`

### Task 5: Portfolio Page Integration
**File:** `src/app/(dashboard)/portfolio/page.tsx`

- Add new "Аналитика" tab or section below existing content
- Fetch analytics data via `getCorrelationMatrixAction` + `getPortfolioAnalyticsAction`
- Layout: 2-column grid
  - Left: Correlation heatmap (full width on mobile)
  - Right top: Sector donut
  - Right bottom: Asset type chart
  - Below: Trade success
