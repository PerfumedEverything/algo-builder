## Goal

## Tasks

### Task 1: Fix paper portfolio price fetching
**Files:** `src/server/actions/operation-actions.ts`

**Root cause:** `getPaperPortfolioAction` uses ONLY `PriceCache.getPrice()`. If cache empty → `currentPrice = undefined` → `unrealizedPnl = 0`.

**Fix:**
1. Before the `for` loop: create `BrokerService` instance, collect unique instruments
2. Batch-fetch prices: `broker.getInstrumentPrice(userId, inst)` with cache fallback
3. Build `prices: Record<string, number>`, use in loop

### Task 2: Fix TopMovers click
**Files:** `src/app/(dashboard)/terminal/page.tsx`

**Root cause:** `handleQuickSelect` calls `getInstrumentsAction("STOCK")` → ticker not found → silent return.

**Fix:** Construct `BrokerInstrument` directly (like `handlePositionSelect` does):
```typescript
const handleQuickSelect = useCallback((t: string) => {
  const inst: BrokerInstrument = {
    figi: "",
    ticker: t,
    name: t,
    type: "STOCK",
    currency: "rub",
    lot: 1,
  }
  handleInstrumentSelect(inst)
}, [handleInstrumentSelect])
```
No async API call needed — candle loading uses ticker, not figi.
