**Gathered:** 2026-03-24
**Revised:** 2026-03-24 — TradingView embed widget doesn't support MOEX free. Switching to lightweight-charts.
**Status:** Ready for replanning

<domain>
## Phase Boundary

Redesign the /terminal page into a professional trading terminal per Anton's wireframe. Use lightweight-charts (open source) for charts with T-Invest data. Add order book (стакан), open positions with P&L, and trade history panels. All styled natively in the app's dark theme.

**NOT in scope:** Order placement, active orders — future phase.
**REMOVED from scope:** Top gainers/losers — was never in Anton's wireframe.
**ABANDONED:** TradingView embed widget — doesn't support MOEX on free tier.

</domain>

<decisions>
## Implementation Decisions

### Chart
- **D-01:** Use lightweight-charts (already in project, working on current terminal). NOT TradingView embed widget — it doesn't support MOEX data on the free tier.
- **D-02:** Keep existing chart component architecture (`instrument-chart.tsx`, `chart-period-selector.tsx`) and enhance it.

### Layout (per Anton's wireframe 2026-03-24)
- **D-03:** Layout:
  - Top: Price bar (ticker, price, change%, H/L, volume, bid/ask) — data from T-Invest
  - Middle: lightweight-charts candlestick chart (left) + Order book/стакан (right) — side by side
  - Below: Open positions (left) + Trade history (right) — same row
- **D-04:** NO top gainers/losers — not in Anton's wireframe.
- **D-05:** NO TradingView widget — doesn't work with MOEX free.

### Order Book (Стакан)
- **D-06:** Table component showing bid/ask levels from T-Invest getOrderBook API. Use system token (public data). Show spread.

### Data Sources
- **D-07:** All data from T-Invest API via existing broker provider:
  - Price, H/L, change%: from getLastPrices + getCandles
  - Bid/Ask: from getOrderBook top level
  - Volume: from candle data
  - Chart candles: from getCandles (existing)
- **D-08:** Order book polling every 5 seconds
- **D-09:** Price updates via existing price-stream-worker SSE

### Positions & History
- **D-10:** Open positions from existing portfolio data with live P&L
- **D-11:** Trade history for selected instrument from existing operations

### Style
- **D-12:** Native dark theme matching rest of the app. shadcn/ui components. NO iframe or external embeds.

### Claude's Discretion
- Order book depth (5 or 10 levels)
- Exact responsive/mobile layout

</decisions>
