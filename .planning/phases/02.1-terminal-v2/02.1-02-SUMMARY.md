## What Was Built

- **TradingViewWidget** (`src/components/terminal/tradingview-widget.tsx`): Script-injection wrapper for TradingView Advanced Chart. Props: `symbol`, `theme`. Uses `useRef` + `useEffect` with `[symbol, theme]` deps. Full toolbar, MOEX symbols, `Europe/Moscow` timezone. Built-in price bar, indicators, and order book (DOM) from TradingView itself.
- **PositionsPanel** (`src/components/terminal/positions-panel.tsx`): Portfolio positions table with P&L. Columns: Тикер, Кол-во, Ср. цена, Текущая, П/У, П/У %. Accepts `priceUpdates` Map for optional realtime enhancement.
- **TradeHistoryPanel** (`src/components/terminal/trade-history-panel.tsx`): Operations for selected ticker with date, type (Покупка/Продажа), price, quantity, amount. Loading skeleton.
- **Terminal page** (`src/app/(dashboard)/terminal/page.tsx`): Simplified layout per Anton's wireframe. InstrumentSelect top-left, TradingView widget full-width, positions+history grid below. No SSE, no order book polling, no top movers.
