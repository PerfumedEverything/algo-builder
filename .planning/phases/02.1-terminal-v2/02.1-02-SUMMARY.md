## What Was Built

- **TradingViewWidget** (`src/components/terminal/tradingview-widget.tsx`): Script-injection wrapper for TradingView Advanced Chart. Props: `symbol`, `theme`. Uses `useRef` + `useEffect` with `[symbol, theme]` deps. Clears innerHTML on cleanup/remount. Full toolbar, MOEX symbols, `Europe/Moscow` timezone.
- **PriceBar** (`src/components/terminal/price-bar.tsx`): Horizontal bar showing ticker name, realtime price from SSE stream, bid/ask from order book data. Green text when price exists.
- **OrderBook** (`src/components/terminal/order-book.tsx`): Bid/ask depth table (10 levels per side). Asks shown reversed (highest-at-top), spread separator row. Red/green tinted rows per side. Uses shadcn Table.
- **PositionsPanel** (`src/components/terminal/positions-panel.tsx`): Portfolio positions table with live P&L. If `priceUpdates.has(instrumentId)`: uses realtime price for P&L calc. Otherwise falls back to broker data.
- **TradeHistoryPanel** (`src/components/terminal/trade-history-panel.tsx`): Operations for selected ticker with date, type (Покупка/Продажа), price, quantity, amount. Loading skeleton.
- **TopMovers** (`src/components/terminal/top-movers.tsx`): Two-column grid with gainers (TrendingUp icon, green %) and losers (TrendingDown icon, red %).
- **Terminal page** (`src/app/(dashboard)/terminal/page.tsx`): Full rewrite. Removed all lightweight-charts code. Layout: row1 InstrumentSelect+PriceBar, row2 TradingView+OrderBook, row3 positions+history grid, row4 top movers. Order book polls every 5s, top movers every 60s.
