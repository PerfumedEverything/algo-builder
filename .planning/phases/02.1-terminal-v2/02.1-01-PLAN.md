1. Extend `src/server/providers/analytics/moex-provider.ts` with `getTopMovers(topN = 5)` method:
- Add type `TopMoverRaw = { SECID: string; LAST: number | null; LASTCHANGEPRCNT: number | null; VOLTODAY: number | null; HIGH: number | null; LOW: number | null }` and `TopMoverNameRaw = { SECID: string; SHORTNAME: string }` as local types (not exported)
- Cache key: `"moex:top-movers"`, TTL: 60 seconds
- Fetch URL: `${this.baseUrl}/engines/stock/markets/shares/boards/TQBR/securities.json?iss.meta=off&marketdata.columns=SECID,LAST,LASTCHANGEPRCNT,VOLTODAY,HIGH,LOW&securities.columns=SECID,SHORTNAME`
- Parse `json.marketdata` block with `mapColumnsToObject<TopMoverRaw>` (reuse existing helper)
- Parse `json.securities` block with `mapColumnsToObject<TopMoverNameRaw>` for SHORTNAME
- Build name map: `Record<string, string>` from securities SECID->SHORTNAME
- Filter out rows where `LAST === null || LASTCHANGEPRCNT === null`
- Sort by `LASTCHANGEPRCNT` descending
- Take top `topN` as gainers (highest positive), last `topN` reversed as losers (most negative)
- Map to `TopMover` type (import from `@/core/types`): `{ ticker: SECID, shortName: nameMap[SECID], price: LAST, changePct: LASTCHANGEPRCNT, volume: VOLTODAY ?? 0, high: HIGH ?? 0, low: LOW ?? 0 }`
- Return `{ gainers: TopMover[], losers: TopMover[] }`

2. Create `src/server/actions/terminal-actions.ts` with `"use server"` directive:

**getOrderBookAction(figi: string, depth = 10):**
- Import `TinkoffInvestApi` from `tinkoff-invest-api`, `ApiResponse`, `successResponse`, `errorResponse`, `OrderBookData` from types
- Use system token pattern (same as price-stream-worker): `new TinkoffInvestApi({ token: process.env.TINKOFF_SYSTEM_TOKEN! })` — NOT user token, because order book is public exchange data (per RESEARCH Open Question 1 and Pitfall 4)
- Call `api.marketdata.getOrderBook({ instrumentId: figi, depth })`
- Map `bids` and `asks` using `toNumber` helper (inline: `(q) => (q?.units ?? 0) + (q?.nano ?? 0) / 1_000_000_000`) to convert `Quotation` type to number
- Calculate spread: `asks[0]?.price - bids[0]?.price` (or 0 if empty)
- Return `ApiResponse<OrderBookData>`

**getTopMoversAction(topN = 5):**
- Import `MOEXProvider` from `@/server/providers/analytics`
- Call `new MOEXProvider().getTopMovers(topN)`
- Return `ApiResponse<{ gainers: TopMover[], losers: TopMover[] }>`
- No auth required (public market data), but wrap in try/catch with errorResponse

**getOperationsByTickerAction(ticker: string):**
- Auth check via `getCurrentUserId()`
- Import `BrokerService` from `@/server/services`
- Call `new BrokerService().getPortfolio(userId)` to get portfolio
- Find position by ticker match: `portfolio.positions.find(p => p.ticker === ticker)`
- Return the position's `operations` array (type `PositionOperation[]`)
- If no position found, return empty array
- Return `ApiResponse<PositionOperation[]>`
