## Test Files

### src/__tests__/terminal/order-book.test.ts
Tests for order book data transformation:
- `toNumber` helper: Quotation (units + nano) to JavaScript number conversion
- `mapOrderBookResponse`: Maps raw T-Invest SDK response to `OrderBookData` type
- Spread calculation: `asks[0].price - bids[0].price`
- Edge cases: empty arrays, undefined Quotation values

### src/__tests__/terminal/top-movers.test.ts
Tests for MOEX ISS top movers parsing:
- `parseTopMoversResponse`: Joins `marketdata` + `securities` blocks on SECID
- Filters rows with null LAST or LASTCHANGEPRCNT
- Sort order: gainers by highest LASTCHANGEPRCNT, losers by most negative
- topN parameter respected
- Mapping to `TopMover` type shape
