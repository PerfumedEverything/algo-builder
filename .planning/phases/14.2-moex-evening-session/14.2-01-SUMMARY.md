## What Was Built

### Task 1: isMarketOpen() with Evening Session

`src/lib/market-hours.ts` — replaced single range `mskMinutes >= 590 && mskMinutes < 1130` with:
```typescript
const inPreOpenAndMain = mskMinutes >= 590 && mskMinutes < 1120
const inEvening = mskMinutes >= 1120 && mskMinutes < 1430
return inPreOpenAndMain || inEvening
```

Coverage: 09:50-18:40 MSK (pre-open + main) + 18:40-23:50 MSK (evening session).

`src/__tests__/terminal/market-hours.test.ts` — 10 unit tests covering all boundaries including evening session, weekends, pre-open. Full TDD cycle: RED (2 failures) → GREEN (10 passes).
