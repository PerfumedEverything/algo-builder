**Step 2: Add periodic stats refresh with immediate first call (SC-3)**

In `src/app/(dashboard)/terminal/page.tsx`, add a new `useEffect` hook that periodically refreshes daily session stats every 60 seconds while an instrument is selected and the market is open.

Insert the following `useEffect` AFTER the existing `handleInstrumentSelect` callback (around line 209), before `handleQuickSelect`:

```typescript
useEffect(() => {
  if (!instrument?.figi) return
  fetchDailyStats(instrument.figi)
  const id = setInterval(() => {
    if (isMarketOpen()) fetchDailyStats(instrument.figi)
  }, 60_000)
  return () => clearInterval(id)
}, [instrument, fetchDailyStats])
```

IMPORTANT: The `fetchDailyStats(instrument.figi)` call BEFORE `setInterval` is critical — it ensures stats are fetched immediately on instrument select during evening session, not after a 60s delay. This matches RESEARCH.md Fix 2 pattern (lines 207-212).
