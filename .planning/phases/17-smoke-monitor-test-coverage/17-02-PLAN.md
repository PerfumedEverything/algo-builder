**scripts/smoke-probes.ts — all 10 exported probe functions:**

```typescript
import "dotenv/config"
import type Redis from "ioredis"
import type { SupabaseClient } from "@supabase/supabase-js"

export type ProbeResult = { name: string; ok: boolean; reason?: string; durationMs: number }
```

Create individual probe functions (all exported for testing):

1. `probeHealth(appUrl: string): Promise<ProbeResult>` — GET `${appUrl}/api/health`, expect 200 + `ok: true` in JSON
2. `probeRedis(redis: Redis): Promise<ProbeResult>` — `redis.ping()`, expect "PONG"
3. `probeDatabase(supabase: SupabaseClient): Promise<ProbeResult>` — `.from("User").select("id").limit(1)`, expect no error
4. `probePriceWorker(redis: Redis): Promise<ProbeResult>` — `redis.get("worker:heartbeat")`, check age < 180_000ms
5. `probeBybitWorker(redis: Redis): Promise<ProbeResult>` — `redis.get("bybit-worker:heartbeat")`, check age < 180_000ms. If key is null, return `{ ok: true, reason: "bybit worker not enabled" }` (not all deploys have Bybit)
6. `probeTelegramBot(redis: Redis): Promise<ProbeResult>` — `redis.get("telegram-bot:heartbeat")`. If no key, try `Bot.api.getMe()` as fallback
7. `probeSignalsCheck(appUrl: string, cronSecret: string): Promise<ProbeResult>` — POST `${appUrl}/api/signals/check` with Bearer header, expect 200
8. `probePricesEndpoint(appUrl: string): Promise<ProbeResult>` — GET `${appUrl}/api/prices/stream`, expect 401 (proves endpoint is alive without auth)
9. `probeCandleCache(redis: Redis): Promise<ProbeResult>` — `redis.keys("candles:*")`, expect at least 1 key
10. `probeActiveStrategies(supabase: SupabaseClient): Promise<ProbeResult>` — query Strategy table for status=ACTIVE count > 0 (SMOKE-04: proves data layer + active trading config is working)

**scripts/smoke-monitor.ts — thin entry point (~40 lines):**

```typescript
import "dotenv/config"
import Redis from "ioredis"
import { Bot } from "grammy"
import { createClient } from "@supabase/supabase-js"
import {
  probeHealth, probeRedis, probeDatabase, probePriceWorker,
  probeBybitWorker, probeTelegramBot, probeSignalsCheck,
  probePricesEndpoint, probeCandleCache, probeActiveStrategies,
  type ProbeResult,
} from "./smoke-probes"
```

Main function `runSmoke()`:
- Instantiate redis, supabase admin client, grammy Bot from env vars
- Run all probes with `Promise.allSettled` + 10s timeout per probe
- Collect results
- If ANY failed: format message with all failures and send via `bot.api.sendMessage(SMOKE_CHAT_ID, message, { parse_mode: "Markdown" })`
- Log results to console regardless

Env vars used: `NEXT_PUBLIC_APP_URL`, `REDIS_URL`, `TELEGRAM_BOT_TOKEN`, `SMOKE_CHAT_ID` (fallback to `TELEGRAM_ADMIN_CHAT_ID`), `CRON_SECRET`, `NEXT_PUBLIC_SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`

Entry point: `runSmoke().then(() => process.exit(0)).catch(() => process.exit(1))`
